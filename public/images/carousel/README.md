Place your carousel images here with the following naming convention:

1.jpg
2.jpg
3.jpg
4.jpg
5.jpg
6.jpg
7.jpg

Recommended size: 1200 x 800 (JPEG or WEBP). If a local image is missing the carousel will automatically fall back to the remote Unsplash image provided in `src/app/page.tsx`.

To use your images:
- Copy your files into `public/images/carousel/`
- Restart the dev server (for local testing):

```bash
npm run dev
```

The carousel will start rotating when the section scrolls into view.
